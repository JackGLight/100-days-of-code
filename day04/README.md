Day 4 – Rock Paper Scissors

A command-line Rock Paper Scissors game where the user plays against the computer.

The program uses randomization to generate the computer’s move, ASCII art to
visualize each choice, and conditional logic to determine the winner of each round.

Concepts practiced:
- lists and indexing
- random number generation
- user input validation
- functions and control flow
- multi-line strings for ASCII graphics

Run:
python main.py
